# Express

這邊使用 MVC 架構可以簡單建構出一個完整的伺服器專案，
在 request 傳給 controller 後，他會傳給 model 來回傳得到資料，再傳給 views 渲染畫面後再回傳，再傳給使用者。

這邊使用 ejs 來進行畫面的修改跟渲染。
